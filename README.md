# Snake Game

A classic Snake game implemented in Python using the Pygame library.

## Features
- 🐍 Snake movement with arrow keys
- 🍎 Food spawning at random locations
- 📈 Increasing speed as the score grows
- 💥 Collision detection (self-collision and wall collision)
- 🎯 Score tracking
- 🎮 Game over screen

## Prerequisites
- Python 3.6+
- Pygame library

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/lavyz-100-Prog/Snake
   cd Snake
   ```

2. Install Pygame:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the game from the terminal:

```bash
python snake.py
```

## Controls
- **Arrow Keys**: Move the snake (Up, Down, Left, Right)
- **R**: Restart the game after a game over
- **Q**: Quit the game

## Game Rules
- Eat the red food to grow longer
- Avoid hitting the walls
- Avoid hitting your own tail
- The game ends when you collide with a wall or your tail
- The speed increases as your score increases
